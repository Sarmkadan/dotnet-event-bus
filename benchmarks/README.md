## Benchmark Categories
- Core Operations: Event publishing throughput
- Parallel Processing: Concurrent handler performance  
- Error Handling: Retry and DLQ performance
- Request-Reply: Synchronous pattern performance
- Batch Operations: Batch publishing efficiency
- Middleware: Cross-cutting concern overhead
- Configuration: Different setup scenarios

- Latency (μs)
- Memory Allocations (B)
